# 💳 Credit Card Fraud Risk Analysis Dashboard

An interactive **Power BI dashboard** designed to analyze credit card transactions, detect fraud patterns, and provide actionable insights for risk monitoring and decision-making.

---

## 📊 Project Overview

This project focuses on identifying fraudulent activities by analyzing transaction data across multiple dimensions such as fraud type, risk level, geography, and time trends.

The dashboard enables users to:
- Detect high-risk transactions  
- Monitor fraud trends over time  
- Understand key fraud drivers  
- Support data-driven financial decisions  

---

## 🎯 Objectives

- Analyze fraud patterns across different transaction types  
- Identify high-risk categories and regions  
- Track key fraud-related KPIs  
- Build an interactive and user-friendly dashboard  

---

## ⚡ Key Features

### 🔹 KPI Overview
- Fraud Rate (%)  
- Total Fraudulent Transactions  
- Critical Risk Transactions  
- Total Fraudulent Amount  

### 🔹 Fraud Type Analysis
- Card Not Present  
- Card Skimming  
- Identity Theft  
- Account Takeover  
- Phishing  

### 🔹 Risk Distribution
- Low Risk  
- Medium Risk  
- High Risk  
- Critical Risk  

### 🔹 State-wise Insights
- Identify top states with highest fraud activity  

### 🔹 Monthly Trends
- Track fraud patterns across months  

---

## 🎨 Dashboard Highlights

- Clean and modern **dark-themed UI**  
- Interactive filters (Fraud Type, State, Merchant Name)  
- Combination of bar charts, donut charts, and line charts  
- Insight-driven layout for quick decision-making
